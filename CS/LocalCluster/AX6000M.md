# ipTIME AX6000M SOP

## 접속

1. 외부 WAN을 노란색에 연결, LAN을 주황색에 연결해야한다
2. `netstat -nr | grep default`
  - `en0`은 와이파이, 설정 혼선을 피하기 위해 off후 설정 진행
  - `en5` c타입 랜선 어댑터를 이용해 연결시 `en0`이 아닌 다른 곳으로 잡히게됨
3. `http://192.168.0.1/ui/`로 어드민 페이지 접속
  - `http://192.168.0.1/ui/dhcpd`페이지에서 DHCP 서버 설정을 통해 NIC에는 고정된 IP를 할당하도록 저장
4. ssh 연결: `ssh gunha@192.168.0.3`
