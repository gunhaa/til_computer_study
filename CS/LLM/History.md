# LLM의 발전과정

- 현대의 LLM은 매우 복잡한 아키텍처로 이루어져있다
- 가장 효율적으로 공부하는 방법은 시간을 거슬러 올라가 그 기술의 핵심 아이디어가 아주 이해하기 쉬운 형태일때 어떤 모습이였는지를 보는 것이다

## LLM의 역사

1. 정보 이론의 시작(1948)
- 클로드 섀넌의 논문 `A Mathematical Theory of Communication`에서 마르코프 체인을 언어에 적용하여 "앞에 나온 글자를 바탕으로 다음 글자의 확률을 결정하는 방식"을 수학적으로 정리했다
- 이 이론이 현대 LLM의 핵심 훈련 방식인 `다음에 올 내용을 예측한다`는 아이디어이다
  - 현대 LLM은 `Next Token Prediction`으로 훈련되며, Context가 많을수록 다음에 올 내용의 확률을 더 정확하게 추정할 수 있다
2. 규칙 기반 시도(1950s - 1970s)
- 문법 규칙을 프로그래밍하는 방식으로 자연어 처리를 시도, 많은 노력에도 실용적인 결과를 내지 못함
3. 통계적 언어 모델(1970s 후)
- Jelinek의 IBM 연구팀이 음성 인식에 통계적 방법을 도임하고 n-gram을 적용
- 규칙 -> 데이터 기반으로 전환
4. 뉴럴 네트워크 모델(2003)
- Bengio et al.이 뉴럴 네트워크로 언어 모델을 만드는 방법을 제시
5. 트랜스 포머(2017)
6. LLM 대중화(2022)
- ChatGPT 공개(2022.11)
- 모델 규모의 데이터를 키우면 성능이 향상된다는 접근
7. AI 에이전트(2024~)
- 강화학습으로 도구 사용 능력 향상
