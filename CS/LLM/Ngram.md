# N-gram 언어 모델

- 언어 모델이란?
  - 언어 모델은 다음에 올 수 있는 단어에 확률을 부여한다
  - ChatGPT에게 같은 질문을 해도 매번 대답이 달라지는 이유이다
- n개의 gram이라는 의미
  - n개의 단어를 사용하는 언어 모델
  - `gram`은 빈칸으로 구분된 단어를 의미한다
  - `n`은 (n-1)개의 단어가 앞에 올 수 있고(입력), 이를 가지고 다음 단어(출력)를 예측하는 것이다
    - 창섭이는 사과를(입력, 2-gram) -> 예측기 -> 좋아해(출력, 1-gram)
- 출시된지 80년된 기술이다

## N-gram이 예측하는 방법

- 입력 - 출력 이 있는 자료구조가 있고, 이 자료구조에 입력에 따른 출력 확률이 적혀있다

## N-gram이 이 시점에 필요할까?

1. 최신 LLM에도 사용한다(데이터 전처리/반복 억제/RAG)
2. 데이터에 대한 감각이 생김(데이터 -> 확률 분포)
3. 트랜스포머도 쉬워진다(최신 LLM도 통계모델)

## N-gram

- 유니그램(Uni-Gram), 바이그램(Bi-Gram), 트라이그램(Tri-Gram), 엔그램(N-Gram) 으로 불린다
  - 유니그램은 한 단어의 예측, 바이 그램은 두 단어, 트라이그램은 세 단어의 예측, 엔그램은 N 단어의 예측을 하는 것이다
    - 한 글자로 gram을 잡고 생성시 n을 높여서 데이터를 만들수록 더 좋은 문장이 만들어진다(n이 높아지면 더 나빠질 수 있다)
      - 하지만 n을 무한대로 늘릴 수는 없다
    - 형태소로 gram을 잡고 생성시 한 글자를 gram으로 잡을때보다 훨씬 자연스럽다
      - 4-gram에서 문장 단위로 자연스럽다
      - 형태소 분석기를 사용한다(조사를 자연스럽게 변환)

## 결론

1. n-gram은 앞에 올 단어로 다음 단어를 예측하는 언어모델이다
2. 데이터의 등장 빈도를 이용해서 표를 만든다
3. n이 증가할수록 성능은 좋아지지만 메모리 사용량이 급격히 늘어난다
4. 데이터에 없는 패턴은 알지 못한다
